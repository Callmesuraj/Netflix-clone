# 🎬 Netflix Clone

A visually appealing Netflix clone built using HTML and CSS.  
This project replicates the UI of Netflix's homepage.

## 🚀 Features
- Responsive design
- Hero section with background image
- Movie cards layout
- Clean UI

## 🛠️ Tech Stack
- HTML
- CSS
